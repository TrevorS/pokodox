# POKODOX

Pokodox: https://pokodox.herokuapp.com/

###### Usage

Status:
```bash
$ curl https://pokodox.herokuapp.com
# {"status":"ok","count":151}
```

Query:
```bash
$ curl https://pokodox.herokuapp.com/pokez/5
# {"id":5,"name":"Charmeleon","url":"http://img.pokemondb.net/artwork/charmeleon.jpg"}
```
